第二章 css 一、css样式:position定位 1.position:relative; 相对定位 参照物:他自己原本位置（标签本来应该在位置） 位置:占有自己原本位置(离家留坑) ------------------------------------ 2.position:absolute; 绝对定位 参照物:默认是浏览器首屏窗口，需要自己指定 给父级设定position:relative; 若多个父级，都有定位，采取就近原则 位置:不占位置（是飘在上面的） ------------------------------------- 3.position:fixed; 固定定位 参照物:浏览器窗口(不会被滚动条拖走) 位置:不占位置（是飘在上面的） 辅助样式:位移 top:20px/-20px; bottom:20px/-20px; left:20px/-20px; right:20px/-20px; 纵深: z-index:数值(9);
